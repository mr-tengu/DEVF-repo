Fundamentos de HTML

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. Utiliza etiquetas para estructurar y presentar contenido en navegadores.

Anatomía de un elemento HTML

Un elemento HTML consta de una etiqueta de apertura, contenido y una etiqueta de cierre. Por ejemplo:

<h1>Título Principal</h1>

En este caso,<h1> es la etiqueta de apertura, "Título Principal" es el contenido y </h1> es la etiqueta de cierre.

HTML Semántico

HTML semántico utiliza etiquetas que describen claramente el propósito del contenido. Ejemplos de etiquetas semánticas incluyen <header>, <nav>, <main>, <section>, <article>, <aside>, y <footer>.

Ejemplos de etiquetas semánticas

  • <header>: Define el encabezado de una página o sección.
  • <nav>: Define un bloque de navegación.
  • <main>: Define el contenido principal de un documento.
  • <section>: Define una sección genérica de un documento.
  • <article>: Define un artículo independiente en un documento.
  • <aside>: Define contenido relacionado pero separado del contenido principal.
  • <footer>: Define el pie de página de una página o sección.
El uso de etiquetas semánticas mejora la accesibilidad y el SEO de tu sitio web, ya que los motores de búsqueda y las tecnologías de asistencia pueden entender mejor la estructura y el contenido de tu página.
