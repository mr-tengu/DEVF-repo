Fundamentos de HTML

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. Utiliza etiquetas para estructurar y presentar contenido en navegadores.

Anatomía de un elemento HTML

Un elemento HTML consta de una etiqueta de apertura, contenido y una etiqueta de cierre. Por ejemplo:

<h1>Título Principal</h1>

En este caso,<h1> es la etiqueta de apertura, "Título Principal" es el contenido y </h1> es la etiqueta de cierre.

Los atributos son parte de la etiqueta de apertura y proporcionan información adicional sobre el elemento. Por ejemplo, en el siguiente código, el atributo class="nota" se utiliza para asignar una clase CSS al elemento <div>
